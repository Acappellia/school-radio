# 需求模块

## 电台调频

- 电台前有一个椅子，点击后玩家坐到椅子上（无法移动）
- 电台有两个旋钮，用w和s切换旋钮，a和d调节旋钮，一个粗调，一格精调，调节频率
- 频率会显示在电台的屏幕上
- 调到正确的频率之后会有反应
